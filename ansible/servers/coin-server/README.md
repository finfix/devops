# Последовательность настройки сервера Coin

1. Установка пользователей на сервер
```sh
make init-users
```

2. Общая настройка сервера для торгов
```sh
make setup-server
```
- Настройка необходимого файлового окружения проекта
- Установка докера
- Настройка необходимого окружения в докере и авторизация в Github
- Установка Postgresql
- Установка Nginx
- Настройка Nginx
- Установка сервера Coin
- Установка тестового окружения Coin
